# Project 1: EC2 Web Server Deployment (Apache)

## Goal
Launch an EC2 instance, connect securely, install Apache, and deploy a public website.

## AWS Services Used
- EC2
- VPC / Subnet
- Security Groups
- AWS CloudShell
- (Linux) Apache HTTP Server

## What I Did
- Launched a Linux EC2 instance
- Connected via SSH
- Installed and managed Apache
- Configured security group rules (HTTP/SSH)
- Deployed a public website and verified it in the browser

## Issues & Fixes
- Debugged SSH connectivity (timeouts)
- Used CloudShell to bypass local network/hotspot issues
- Corrected network/security rules to allow secure access
