# AWS Project 2: EC2 Web Server with EBS Storage

## Overview
This project demonstrates launching a cloud server and attaching persistent storage using AWS.

## Services Used
- Amazon EC2
- Amazon Elastic Block Store (EBS)
- Security Groups
- NGINX

## Architecture
User → EC2 Instance → EBS Volume

## Steps Completed
1. Launched EC2 instance
2. Installed NGINX web server
3. Configured HTTP access through security groups
4. Created and attached EBS volume
5. Formatted and mounted storage
6. Verified persistent storage functionality

## Key Concepts
- Block storage
- Linux disk mounting
- Persistent cloud storage
